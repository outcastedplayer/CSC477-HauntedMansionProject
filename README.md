# CSC477-HauntedMansionProject
A 3D puzzle game set in a Haunted Mansion


# Ideas
Ghost - implement a ghost that walks between rooms

Hiding Spots - places to enter to prevent ghost from killing you (under tables, closets, others)


# Puzzles
Puzzle box on table with key to unlock a door - William

Graveyard with certain graves you need to dig up to unlock the front door (Final puzzle to escape) - Carl

Chinese door problem (have a language book and need a specific english word to unlock door) - Bryce
